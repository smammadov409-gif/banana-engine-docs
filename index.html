API Documentation

Complete guide for the Banana Engine v1.9 Ultra API functions.

Banana.Init(w, h)

Initializes the engine with specific width and height.

Banana.Clear()

Clears the current frame for the next render call.

Banana.DrawJoy(bx, by, br, c)

Renders a virtual joystick at coordinates with color.

Banana.LoadModel(url)

Returns a Model object from a given image/sprite URL.

Banana.DrawRect(x,y,w,h,c)

Draws a filled rectangle on the canvas.

Banana.SetShadow(x,y,b,c)

Configures global shadow offset, blur, and color.

Banana.Shake(intensity)

Triggers a camera shake effect with given power.

Banana.Vibrate(ms)

Sends a vibration pulse to supported mobile devices.

Banana.CheckCol(r1, r2)

Checks intersection between two rectangular objects.

Banana.StorageSet(k, v)

Stores data key-value pair in the browser memory.


Quick Loop Example:

Banana.Init(1280, 720);
function loop() {
  Banana.Clear();
  Banana.DrawText("Dragon Bite Active", 100, 100, 30, "yellow");
  Banana.DrawJoy(150, 500, 60, "rgba(255,255,255,0.2)");
  requestAnimationFrame(loop);
}
loop();

Start Your Project

Open BananaEngine.exe. No installation required. Use the built-in Monaco editor to write your logic using the API documented in the "Docs" tab.
