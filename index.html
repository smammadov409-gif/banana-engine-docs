Welcome to Banana Engine

Banana Engine 2D is a high-speed, lightweight game engine for the web. It is built by Dighmations Studio to make game development easy and fast, even on weak computers.

Top Features

  • Ultra Fast: Low CPU usage, high FPS.
  • Mobile Ready: Built-in Joystick and Vibrate.
  • Simple API: You don't need to be a pro to make a game.

Banana.Init()

This function starts the engine and sets your game screen size.

Banana.Init(800, 450);

It creates the canvas and prepares the engine for drawing.

The Main Loop

The loop runs your game at 60 frames per second. Everything that moves must be inside this loop.

function loop() {
    Banana.Clear();
    // Your game code here
    requestAnimationFrame(loop);
}
loop();

Banana.Clear()

Banana.Clear() is the most important command for a clean game. It wipes the screen every frame so your player doesn't leave a "trail" behind while moving.

Banana.Clear();

Why Use It?

In a 2D engine, "moving" an object actually means drawing it in a new position. Without Clear(), the old drawings stay on the screen. Always put this at the top of your Loop.

Banana.Vibrate()

This command makes the player's phone vibrate. It is perfect for explosions, taking damage, or collecting items.

Banana.Vibrate(100);

How to Use

The number inside the brackets is milliseconds (1000 = 1 second). Use a small number like 50 for a light hit, and 500 for a big explosion.

Banana.CheckCol()

Checks if two boxes are touching each other.

if (Banana.CheckCol(player, enemy)) {
    // Game over!
}

Banana.DrawJoy()

Draws a mobile joystick on the screen for player movement.

Banana.DrawJoy(100, 300, 50, "yellow");

Basic Game Code

Copy this to start your first game!

Banana.Init(800, 400);
let p = { x: 50, y: 50, w: 40, h: 40 };

function loop() {
    Banana.Clear();
    if(Banana.TouchDown()) p.x += 5;
    Banana.DrawRect(p.x, p.y, p.w, p.h, "white");
    requestAnimationFrame(loop);
}
loop();
